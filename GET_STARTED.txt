QUICK START (5 minutes):

1. Install Godot 4.0+ from https://godotengine.org/

2. Open the project:
   - Launch Godot
   - Click "Open Project"
   - Navigate to: godot_game/
   - Open "project.godot"

3. Run the game:
   - Press F5 or click the Play button
   - Game window opens with test scene

4. Test the controls:
   - A/D: Move left/right
   - Space: Jump
   - [: Light attack
   - ]: Heavy attack
   - Shift: Block
   - Ctrl: Dodge
   - Plug in controller for full gamepad support
